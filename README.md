 🧮 Simple Calculator UI

A clean and responsive **calculator user interface** built using **HTML and CSS**. This project focuses on layout design and structure, making it ideal for beginners learning front-end basics.


 🚀 Features

* Calculator layout with buttons and display
* Includes operators: +, -, *, /, %
* Clear (AC) and Delete (DEL) buttons
* Decimal input support
* Simple and user-friendly design
* Responsive structure



 🛠️ Technologies Used

* HTML5
* CSS3



 📁 Project Structure

calculator/
│── index.html
│── index.css


▶️ How to Use

1. Clone this repository:

   bash
   git clone https://github.com/your-username/calculator.git
